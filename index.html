Welcome to Qreview Pages.

Reviewing the code and listing all the review item in tacking mode, is always important. Most of the scenarios, it is found that people use spreadsheet to track it. We found it less useful thru spreadsheet, so, developed Qreview.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout

If you're using the GitHub for Mac, simply clone review repository to customise it for your need.

Technology Used

We tried to use some of the open source technologies with mixed flavour of technologies which we think we are good at. Following are the list of framework/api...

  1. play 2.1.1

  2. Java

  3. Scala

  4. JQuery

  5. Backbone JS

  6. Bootstrap (for the UI)

Features

Right now, only two features are available for the use,

  • Review Tracking
  • Task Listing

Task listing is the cool one, which can be used for list all the short span todo's which could be always on to remember and list it all together.
